# Simple Image API 🌌

一个极致轻量的自托管随机图片 API 服务。专为拥有个人服务器/VPS 的用户设计，使用 FastAPI 构建，支持 Docker 一键部署。

**告别繁琐的数据库和后台面板！** 你只需要把图片扔进文件夹，剩下的自动分类、横竖屏识别、缓存刷新全都交由程序自动完成。

## ✨ 核心特性

* **📁 文件夹即分类**：在 `images` 目录下创建子文件夹（如 `anime`, `dota2`），系统自动生成对应的 API 路由。
* **📐 自动横竖屏识别**：自动读取图片比例，横图归类为 `pc`，竖图归类为 `mobile`，无需手动打标签。
* **⚡ 毫秒级热重载**：内置文件系统监控，通过 SFTP/宝塔等方式上传或删除图片后，API 缓存会自动秒级刷新。
* **🪶 极低资源占用**：非常适合部署在甲骨文 ARM 免费机或内存较小的 VPS 上。

## 🚀 目录结构说明

在使用前，请按照以下结构组织你的图片文件夹。API 会自动将文件夹名称（小写）注册为分类名称：

```text
images/
  ├── anime/         <-- 将注册为 /api/anime
  │   ├── 1.jpg      (宽>=高，自动识别为 pc)
  │   └── 2.png      (宽<高，自动识别为 mobile)
  └── dota2/         <-- 将注册为 /api/dota2
      └── ...
```

## 🔗 API 调用规则

服务默认运行在 `8000` 端口。假设你的服务器 IP 或域名为 `http://your-server-ip:8000`。

### 1. 基础全局接口
从图库的所有图片中进行随机抽取。

| 接口地址 | 适用场景 | 返回结果说明 |
| :--- | :--- | :--- |
| `/api/random` | 全局随机 | 库中所有图片的随机一张 |
| `/api/pc` | 电脑壁纸 | 库中所有横屏图片的随机一张 |
| `/api/mobile` | 手机壁纸 | 库中所有竖屏图片的随机一张 |

### 2. 分类指定接口
仅从你指定的文件夹（分类）中进行随机抽取。

| 接口地址 | 适用场景 | 返回结果说明 |
| :--- | :--- | :--- |
| `/api/{分类名}` | 垂类展示 | 抽取指定文件夹内的任意一张图片 |
| `/api/{分类名}/pc` | 垂类电脑壁纸 | 抽取指定文件夹内的横屏图片 |
| `/api/{分类名}/mobile` | 垂类手机壁纸 | 抽取指定文件夹内的竖屏图片 |

> **💡 调用示例：**
> * 获取一张二次元电脑壁纸：`http://your-server-ip:8000/api/anime/pc`
> * 获取一张 Dota2 随机图片：`http://your-server-ip:8000/api/dota2`

## 🐳 快速部署 (Docker Compose)

只需拉取代码，把图片放进 `images` 文件夹，然后在项目根目录执行：

```bash
services:
  image-api:
    image: yuwancumian2009/randompicapi:latest
    container_name: image-api
    restart: unless-stopped
    ports:
      - "8000:8000"
    volumes:
      - ./images:/app/images
```

## 注意事项：

支持的图片格式为：.jpg, .jpeg, .png, .gif, .webp。

建议文件夹名称使用英文或数字，避免在部分不支持中文 URL 编码的客户端中请求失败。保留关键字 pc, mobile, random 不可用作文件夹名。
